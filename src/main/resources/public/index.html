Part 2

  • 1. Get Postman: Link

  • 2. Send a POST request to /hash with the request body in the form of ```{"text": "xyz"}``` (swap "xyz" for your preferred string)

  • 3. Check the result in Postman

Second way: Curl

  • 1. Install curl if you don't have it yet: Link

  • 2. Paste the command below to the terminal (swap "xyz" for your preferred string)

    • curl --header "Content-Type: application/json" --request POST --data '{"text":"xyz"}' http://localhost:4567/hash

  • 3. Check the result in the terminal
